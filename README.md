# gonesis
Generate Golang project template ready to be pushed on GitHub with a single command (Go + Genesis)

Installation 📡
-------
**Go 1.17+**
```bash
go install -v github.com/edoardottt/gonesis@latest
```
**otherwise**
```bash
go get -v github.com/edoardottt/gonesis
```

Usage 💻
-------
```console
gonesis
``` 
or if you encounter errors try:
```console
~/go/bin/gonesis
```
