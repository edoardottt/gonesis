# gonesis
Create Golang project structure ready to be pushed on GitHub

Installation 📝
-------
**Go 1.17+**
```bash
go install -v github.com/edoardottt/gonesis@latest
```
**otherwise**
```bash
go get -v github.com/edoardottt/gonesis
```
